## Getting started

```bash
git clone https://github.com/elenakabachenko/rn-gallery
cd ./rn-gallery
npm i
```

### Run IOS:
```bash
react-native run-ios
```

### Run Android:
```bash
react-native run-android
```


If you want to use your own `CLIENT_ID` please change it in `./config/application.js`
